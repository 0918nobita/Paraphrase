  2. インストール方法
    1. ビルド済みバイナリの場合：
        インストール作業は不要です。 強いて言えば、適当なディレクトリに展開し、 生成された para ファイル（Windows の場合は para.exe）に パスを通せば終わりです。
    2. ソースからビルドする場合
        5 にも挙げてある  paraphrase/BUILD.txtを参考に、 ソースコードから para ファイル（Windows の場合は para.exe）を ビルドして下さい。 無事にビルドできたら、para にパスを通して下さい。
